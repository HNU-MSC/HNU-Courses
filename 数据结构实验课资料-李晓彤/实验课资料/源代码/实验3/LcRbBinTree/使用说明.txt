编译环境：DEVc++ 
功能：
1、判断是否是空树
2、层次遍历
3、层次遍历的结点对应的父亲节点
4、层次遍历的结点对应的左孩子结点
5、层次遍历的结点对应的右兄弟结点
6、树的深度（层数从0开始算起）
7、树的结点个数

输入说明：以字符串形式输入要构建的二叉树时，按层次遍历的顺序输入，所有的空节点以/代替。注意！必须要输入左子树

测试样例一：
输入:ABCDEF
输出：
判断树是否是空树：不是空树
层次遍历：A B C D E F
层次遍历的结点对应的父亲结点为：
/ A A B B C

层次遍历的结点对应的左孩子结点为：
B D F / / /

层次遍历的结点对应的右兄弟结点为：
/ C / E / /

树的深度：2
树的结点个数：6

测试样例二：
输入：/
输出：判断树是否是空树：输入为一棵空树,程序结束

测试样例三：
输入：AB/CD
输出：
判断树是否是空树：不是空树
层次遍历：A B C D
层次遍历的结点对应的父亲结点为：
/ A B B

层次遍历的结点对应的左孩子结点为：
B C / /

层次遍历的结点对应的右兄弟结点为：
/ / D /

树的深度：2
树的结点个数：4

测试样例四：
输入：AB
输出：
判断树是否是空树：不是空树
层次遍历：A B
层次遍历的结点对应的父亲结点为：
/ A

层次遍历的结点对应的左孩子结点为：
B /

层次遍历的结点对应的右兄弟结点为：
/ /

树的深度：1
树的结点个数：2

测试样例五：
输入：ABC
输出：
判断树是否是空树：不是空树
层次遍历：A B C
层次遍历的结点对应的父亲结点为：
/ A A

层次遍历的结点对应的左孩子结点为：
B / /

层次遍历的结点对应的右兄弟结点为：
/ C /

树的深度：1
树的结点个数：3
